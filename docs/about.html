Governance You Can Verify

OpenResPublica TruthChain is a pilot initiative by {{LGU_NAME}} to modernize local governance using cryptographic principles. Every official document issued carries a permanent, tamper-proof fingerprint anchored to an immutable ledger — visible to all, alterable by none.

🔐

Immutable Ledger

Every document hash is stored in immudb. Once recorded, it cannot be deleted or modified by anyone — not even system administrators — ensuring total accountability.

🧮

Zero Trust Verification

We don't ask you to trust a person. We ask you to trust the math. SHA-256 fingerprinting and Ed25519 cryptography ensure every document's authenticity is provable.

🌐

Public Audit Portal

All records are published to a public GitHub ledger. Any citizen, any time, anywhere in the world can independently verify any document issued by this office.

How It Works
1

Document is Fingerprinted

The original PDF is hashed using SHA-256 — producing a unique 64-character fingerprint. Any alteration, even a single pixel, produces a completely different hash.

2

Authority Signs the Hash

The Punong Barangay's office signs the fingerprint using Ed25519 elliptic curve cryptography — mathematically binding the document to this office.

3

Anchored to immudb

The hash is written to immudb — an immutable, append-only database. The transaction ID serves as permanent proof of the exact time of issuance.

4

Published to Public Ledger

Metadata is committed to this GitHub portal within 90 seconds of stamping. Citizens can verify any document by scanning its QR code or entering its fingerprint.
